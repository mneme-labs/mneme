# Pontus — MnemeCache Rust Client Library

Pontus is the official Rust client library for MnemeCache. It provides:

- **`MnemePool`** — connection pool with health checks, min-idle warm-up, and HA failover
- **`MnemeConn`** — single multiplexed TLS connection with req_id dispatch
- **`Pipeline`** — batch multiple commands in one write (zero extra round-trips)
- Full command surface — String, Hash, List, Sorted Set, Counter, JSON, Admin, DB namespace
- MONITOR streaming subscription
- RBAC-aware auth (token or username/password)

---

## Adding to Your Project

```toml
# Cargo.toml
[dependencies]
mneme-client = { path = "../mneme-client" }
mneme-common = { path = "../mneme-common" }   # Value type, CmdId, etc.
tokio        = { version = "1", features = ["full"] }
anyhow       = "1"
rmp-serde    = "1"
```

## Authentication

MnemeCache requires a session token on every connection. Obtain one via the CLI:

```bash
TOKEN=$(mneme-cli -u admin -p secret auth-token)
echo $TOKEN
```

Or generate one programmatically in your Rust code:

```rust
// Connect without auth, then authenticate with credentials to get a token
// (use mneme-cli or the admin API for initial bootstrap)
```

## Connection Pool

```rust
use mneme_client::{MnemePool, PoolConfig, Consistency};
use std::time::Duration;

let pool = MnemePool::new(PoolConfig {
    // Primary Core address
    addr: "127.0.0.1:6379".into(),

    // Additional Core addresses for HA failover
    // Tried in order if the primary is unreachable or returns LeaderRedirect
    addrs: vec![
        "core-2:6379".into(),
        "core-3:6379".into(),
    ],

    // Path to the CA certificate (generated by `auto_generate = true`)
    // None = insecure mode (dev only — skips TLS cert verification)
    tls_ca_cert: Some("/etc/mneme/ca.crt".into()),

    // Must match the CN or SAN in the server certificate
    // Same as config.tls.server_name on the Core node
    server_name: "mneme.local".into(),

    // Session token (from auth-token CLI command)
    token: std::env::var("MNEME_TOKEN").unwrap_or_default(),

    // Pool sizing
    min_idle:        2,
    max_size:        20,
    acquire_timeout: Duration::from_millis(200),
    health_interval: Duration::from_secs(30),
    idle_timeout:    Duration::from_secs(300),
}).await?;

// Acquire a connection (RAII — returned to pool on drop)
let conn = pool.acquire().await?;
```

### Solo mode (no TLS verification)

```rust
let pool = MnemePool::new(PoolConfig {
    addr:        "127.0.0.1:6379".into(),
    tls_ca_cert: None,           // insecure — dev only
    server_name: "mneme.local".into(),
    token:       token.clone(),
    ..Default::default()
}).await?;
```

## String / KV

```rust
use mneme_common::types::Value;

// SET with TTL
conn.set(b"session:abc", Value::String(b"token123".to_vec()), 3_600_000 /* ms */, Consistency::Quorum).await?;

// SET permanent
conn.set(b"greeting", Value::String(b"hello".to_vec()), 0, Consistency::Quorum).await?;

// GET → Option<Value>
if let Some(val) = conn.get(b"greeting").await? {
    if let Value::String(bytes) = val {
        println!("{}", String::from_utf8_lossy(&bytes));
    }
}

// DEL → u64 (count deleted)
let n = conn.del(&[b"k1", b"k2", b"k3"]).await?;

// EXISTS → bool
let found = conn.exists(b"greeting").await?;

// TTL → i64 ms remaining (-1 = no expiry, -2 = not found)
let ms = conn.ttl(b"session:abc").await?;

// EXPIRE — reset TTL
conn.expire(b"session:abc", 7200).await?;

// GETSET — atomic swap, returns old value
let old = conn.getset(b"key", Value::String(b"new".to_vec())).await?;

// Bulk
conn.mset(vec![
    (b"k1".to_vec(), Value::String(b"v1".to_vec()), 0u64),
    (b"k2".to_vec(), Value::String(b"v2".to_vec()), 0u64),
]).await?;

let vals = conn.mget(&[b"k1", b"k2", b"missing"]).await?;
// vals[2] == None
```

## Counter

```rust
// Initialise
conn.set(b"visits", Value::Counter(0), 0, Consistency::Quorum).await?;

// INCR / DECR → i64
let n = conn.incr(b"visits").await?;          // 1
let n = conn.incrby(b"visits", 10).await?;    // 11
let n = conn.decr(b"visits").await?;          // 10
let n = conn.decrby(b"visits", 5).await?;     // 5
let f = conn.incrbyfloat(b"rate", 0.5).await?; // f64
```

## Hash

```rust
// HSET — multiple field-value pairs (raw bytes)
conn.hset(b"user:1", vec![
    (b"name".to_vec(), b"Alice".to_vec()),
    (b"email".to_vec(), b"alice@example.com".to_vec()),
    (b"age".to_vec(), b"30".to_vec()),
]).await?;

// HGET → Option<Vec<u8>>
if let Some(name) = conn.hget(b"user:1", b"name").await? {
    println!("{}", String::from_utf8_lossy(&name));
}

// HDEL → u64 (removed count)
conn.hdel(b"user:1", &[b"age"]).await?;

// HGETALL → Vec<(Vec<u8>, Vec<u8>)>
let fields = conn.hgetall(b"user:1").await?;
for (field, value) in &fields {
    println!("{} = {}", String::from_utf8_lossy(field), String::from_utf8_lossy(value));
}
```

## List

```rust
// Push to head / tail
conn.lpush(b"queue", vec![b"task1".to_vec(), b"task2".to_vec()]).await?;
conn.rpush(b"queue", vec![b"task3".to_vec()]).await?;

// Pop → Option<Value>
let item = conn.lpop(b"queue").await?;
let item = conn.rpop(b"queue").await?;

// LRANGE — negative indices count from tail
let items = conn.lrange(b"queue", 0, -1).await?;  // all items
let items = conn.lrange(b"queue", 0, 9).await?;   // first 10
```

## Sorted Set

```rust
use mneme_common::types::ZSetMember;

// ZADD → u64 (newly added)
conn.zadd(b"leaderboard", vec![
    ZSetMember { member: b"alice".to_vec(), score: 1500.0 },
    ZSetMember { member: b"bob".to_vec(),   score: 1200.0 },
]).await?;

// ZSCORE → Option<f64>
let score = conn.zscore(b"leaderboard", b"alice").await?;

// ZRANK → Option<u64> (0-indexed ascending)
let rank = conn.zrank(b"leaderboard", b"alice").await?;

// ZRANGE — by rank
let top10 = conn.zrange(b"leaderboard", 0, 9, true).await?;
for m in &top10 {
    println!("{}: {}", String::from_utf8_lossy(&m.member), m.score);
}

// ZRANGEBYSCORE — by score range
let mid = conn.zrangebyscore(b"leaderboard", 1000.0, 1499.0).await?;

// ZCARD → u64
let count = conn.zcard(b"leaderboard").await?;

// ZREM → u64 (removed)
conn.zrem(b"leaderboard", &[b"bob"]).await?;
```

## JSON

```rust
// JSON.SET — set entire document
conn.json_set(b"config", "$", r#"{"timeout":30,"retries":3}"#).await?;

// JSON.SET — update a nested field
conn.json_set(b"config", "$.timeout", "60").await?;

// JSON.GET → Option<String>
if let Some(json) = conn.json_get(b"config", "$").await? {
    println!("{json}");
}
if let Some(timeout) = conn.json_get(b"config", "$.timeout").await? {
    println!("timeout = {timeout}");
}

// JSON.TYPE → Option<String> ("object", "array", "string", "number", etc.)
let t = conn.json_type(b"config", "$").await?;

// JSON.EXISTS → bool
let ok = conn.json_exists(b"config", "$.timeout").await?;

// JSON.DEL → u64 (deleted)
conn.json_del(b"config", "$.retries").await?;

// JSON.ARRAPPEND → u64 (new length)
conn.json_arrappend(b"config", "$.tags", r#""fast""#).await?;

// JSON.NUMINCRBY → f64
let v = conn.json_numincrby(b"config", "$.timeout", 5.0).await?;
```

## Database Namespacing

```rust
// Create a named database (admin role)
let db_id = conn.db_create("analytics", None).await?;  // None = auto-assign ID

// List all named databases
let dbs = conn.db_list().await?;

// Switch active database on this connection
conn.select_id(1).await?;
conn.select_name("analytics").await?;

// Count keys in active database
let n = conn.dbsize_id(None).await?;   // None = active db
let n = conn.dbsize_name("analytics").await?;

// SCAN with glob pattern
let mut cursor = 0u64;
loop {
    let page = conn.scan(cursor, Some("user:*"), 100).await?;
    for key in &page.keys {
        println!("{}", String::from_utf8_lossy(key));
    }
    cursor = page.next_cursor;
    if cursor == 0 { break; }
}

// Or scan all pages at once
let all_keys = conn.scan_all(Some("user:*"), 100).await?;

// TYPE of a key
let t = conn.type_of(b"user:1").await?;  // "hash"

// FLUSHDB — delete all keys (admin)
conn.flushdb_id(None, true).await?;       // sync=true replicates tombstones

// Drop a named database (removes name; data remains accessible by ID)
conn.db_drop("analytics").await?;
```

## Pipeline

A `Pipeline` batches all queued commands into a single `write_all()` call,
eliminating per-command TCP round-trips.

```rust
use mneme_client::{Pipeline, Consistency};
use mneme_common::{CmdId, types::Value};

let mut p = Pipeline::new();

// Build the pipeline (methods are chainable)
p.set(b"key1", Value::String(b"v1".to_vec()), 0, Consistency::Quorum)
 .set(b"key2", Value::Counter(42), 3_600_000, Consistency::Quorum)
 .incr(b"key2")
 .get(b"key1")
 .hset(b"user:1", vec![(b"name".to_vec(), b"Alice".to_vec())])
 .hget(b"user:1", b"name");

// Execute: one write, collect all responses in submission order
let results = conn.execute_pipeline(p).await?;

for (i, frame) in results.iter().enumerate() {
    match frame.cmd_id {
        CmdId::Ok => {
            // Decode according to the command at position i
            // set  → () or "OK"
            // incr → i64
            // get  → Value
            // hget → bytes
        }
        CmdId::Error => {
            let msg: String = rmp_serde::from_slice(&frame.payload)?;
            eprintln!("command {i} failed: {msg}");
        }
        _ => {}
    }
}
```

### Pipeline — counter example

```rust
let mut p = Pipeline::new();
p.incr(b"hits")
 .incr(b"hits")
 .incr(b"hits");

let results = conn.execute_pipeline(p).await?;
for frame in &results {
    if frame.cmd_id == CmdId::Ok {
        let n: i64 = rmp_serde::from_slice(&frame.payload)?;
        println!("hits = {n}");
    }
}
```

## MONITOR Stream

Use a dedicated connection for monitoring. Other commands sent on the same
connection while monitoring is active will have undefined behaviour.

```rust
use mneme_client::{MnemePool, PoolConfig};

// Open a separate connection just for monitoring
let pool = MnemePool::new(PoolConfig {
    addr:        "127.0.0.1:6379".into(),
    tls_ca_cert: Some("/etc/mneme/ca.crt".into()),
    server_name: "mneme.local".into(),
    token:       token.clone(),
    ..Default::default()
}).await?;

let conn = pool.acquire().await?;
let mut stream = conn.monitor().await?;

while let Some(event) = stream.next().await {
    // Format: "<timestamp_ms> <command> <key>"
    println!("[MONITOR] {event}");
}
```

Non-blocking poll:

```rust
while let Some(event) = stream.try_next() {
    println!("[MONITOR] {event}");
}
```

## Admin Operations

### User management

```rust
// Create user (admin role required)
conn.user_create("alice", "password123", "readwrite").await?;

// Grant access to specific databases
conn.user_grant("alice", 1).await?;   // db 1
conn.user_grant("alice", 2).await?;   // db 2

// Revoke database access
conn.user_revoke("alice", 2).await?;

// Change role
conn.user_set_role("alice", "readonly").await?;

// Get user info
let info = conn.user_info(Some("alice")).await?;
println!("role: {}, allowed_dbs: {:?}", info.role, info.allowed_dbs);

// Get own info
let me = conn.user_info(None).await?;

// List all users
let names = conn.user_list().await?;

// Delete user
conn.user_delete("alice").await?;
```

### Cluster operations

```rust
// Cluster state
let info = conn.cluster_info().await?;
for (k, v) in &info {
    println!("{k}: {v}");
}

// Slot assignment
let slots = conn.cluster_slots().await?;
for s in &slots {
    println!("slots {}-{} → {}", s.start, s.end, s.addr);
}

// List Keeper nodes
let keepers = conn.keeper_list().await?;
for k in &keepers {
    println!("  {} ({}) — {}/{} bytes", k.name, k.addr, k.used_bytes, k.pool_bytes);
}

// Pool statistics
let stats = conn.pool_stats().await?;
println!("pool {}/{} bytes, {} keepers", stats.used_bytes, stats.total_bytes, stats.keeper_count);

// Memory usage of a specific key
let bytes = conn.memory_usage(b"user:1").await?;

// Slow query log
let log = conn.slowlog().await?;
for entry in &log {
    println!("{} µs — {} {:?}", entry.duration_us, entry.command, entry.key);
}

// Server stats (INFO-style text)
let stats_text = conn.stats().await?;
println!("{stats_text}");

// Generate Keeper join token
let token = conn.generate_join_token().await?;
println!("use: mneme-keeper --join-token {token}");

// Wait for replication durability
let acked = conn.wait(2, 5000).await?;   // wait up to 5s for 2 keepers
assert!(acked >= 2, "replication lagging");

// Hot-reload config
conn.config_set("memory.pool_bytes", "4gb").await?;
```

## Consistency Levels

| Level | Bits | Behaviour | Use when |
|-------|------|-----------|----------|
| `Consistency::Eventual` | `00` | Fire-and-forget; reads from replicas if available | Counters, caches, non-critical reads |
| `Consistency::Quorum` | `01` | `floor(N/2)+1` Keeper ACKs (**default**) | Most writes |
| `Consistency::All` | `10` | Every Keeper ACKs | Maximum durability |
| `Consistency::One` | `11` | First Keeper ACK | Low-latency durable writes |

```rust
conn.set(b"k", Value::Counter(1), 0, Consistency::Quorum).await?;    // default
conn.set(b"k", Value::Counter(1), 0, Consistency::Eventual).await?;  // fastest
conn.set(b"k", Value::Counter(1), 0, Consistency::All).await?;       // most durable
```

## HA Failover

The pool handles leader redirect automatically:

1. If the primary `addr` returns `LeaderRedirect`, the pool reconnects to the
   redirected address and retries.
2. If the primary `addr` is unreachable, the pool tries `addrs` in order.
3. After reconnecting, AUTH is re-sent automatically.

```rust
let pool = MnemePool::new(PoolConfig {
    addr:  "core-1:6379".into(),
    addrs: vec!["core-2:6379".into(), "core-3:6379".into()],
    tls_ca_cert: Some("/etc/mneme/ca.crt".into()),
    server_name: "mneme.local".into(),
    token:       token.clone(),
    ..Default::default()
}).await?;
```

## Error Handling

All methods return `anyhow::Result<T>`. Server errors are surfaced as
`anyhow::Error` with the original error string from the server payload.

```rust
match conn.get(b"missing").await {
    Ok(Some(val)) => { /* found */ }
    Ok(None)      => { /* key not found */ }
    Err(e) if e.to_string().contains("WrongType") => { /* type mismatch */ }
    Err(e) => return Err(e),
}
```
